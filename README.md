# Business Website - React Multi Page App

A professional multi-page React website built with Vite. This is not a React Router SPA. Each page has its own HTML entry file and its own React entry point.

## Pages

- `/` Home
- `/about/` About
- `/services/` Services
- `/portfolio/` Portfolio
- `/contact/` Contact

## Features

- Responsive navigation
- Animated hero and scroll reveal sections
- Testimonial / business carousel
- Random related business image component
- Professional multi-page structure
- Modern CSS with variables and reusable components
- Contact form UI with validation-style feedback

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```
